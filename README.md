#   Algorithms & Data Structures

Each folder contains a self-contained lab exercise covering a core computer science concept in Java.

---

## 📌 About This Repo
This repository contains my lab work for the CS211 Algorithms & Data Structures module at Maynooth University. Labs are added weekly throughout the semester and cover fundamental data structures and algorithmic concepts implemented in Java.

---

## 📁 Structure
Each lab lives in its own folder:

```
├── Lab01/          # Lab topic
├── Lab02/          # Lab topic
├── Lab03/          # Lab topic
└── ...             # More labs added weekly
```

Each folder contains:
- The `.java` source file(s)
- Any supporting files required to run the lab

---

## 🛠️ Built With
- Java
- Object-Oriented Programming
- Core CS concepts — see individual lab folders for details

---

## 🚀 How to Run Any Lab

**Requirements:** Java JDK installed

```bash
# Navigate into the lab folder
cd Lab01

# Compile
javac *.java

# Run
java Main
```

- Searching Algorithms
